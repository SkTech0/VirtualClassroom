# Live Study Room (Frontend)

Angular 19 frontend for VirtualClassroom. For setup, run instructions, and deployment see the **root [README.md](../README.md)**.

- **Dev:** `npm install` then `ng serve` (app at http://localhost:4200).
- **Build:** `ng build` or `npm run build:railway` with `API_URL`/`HUB_URL` for production.
